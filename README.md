# yb

A small CLI that helps manage and setup yubikeys for developers.

## Install

```
brew tap cagiti/tap
brew install yb
```

## Usage

List yubikeys
- `yb list`

**e.g.**

```
‚ùØ yb list
+-----+------------------------------+
| REF | CARD                         |
+-----+------------------------------+
|   0 | Yubico YubiKey OTP+FIDO+CCID |
+-----+------------------------------+
```
